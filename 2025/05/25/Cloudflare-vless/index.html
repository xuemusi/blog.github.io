新手小白如何通过 Cloudflare 搭建永久免费节点，告别流量焦虑？

搭建永久免费节点的好处

  • 自建节点，安全可靠：不依赖第三方机场，避免隐私泄露，安全性更高。
  • 永久免费，不限流量：利用 Cloudflare 的免费服务，突破传统 VPS 或机场的流量限制。

搭建 CF 免费节点需要准备的内容

1. Google 账号

注册一个 Google 账号，用于登录 Cloudflare、GitHub、注册域名等操作。

2. 注册 Cloudflare 账号

访问 Cloudflare 官方网站并注册账号：

👉 https://dash.cloudflare.com/

3. 购买一个域名

建议选择支持自定义 DNS 的域名服务商，例如 Spaceship：

👉 https://www.spaceship.com/zh/domain-search/?query=&tab=domains

4. 生成 UUID

用于配置 VLESS 或 Trojan 节点的唯一标识符：

👉 https://www.wetools.com/uuid

VLESS + Cloudflare Workers 方案（推荐）

👉 GitHub 地址

Trojan + Cloudflare Pages 方案

使用 Pages 部署 Trojan，同样基于 yonggekkk 提供的仓库：

购买域名并绑定到 Cloudflare

  1. 登录域名注册商（如 Spaceship）后台。
  2. 设置域名 DNS 指向 Cloudflare 提供的解析地址。
  3. 在 Cloudflare 后台添加域名，开启代理功能（橙色云图标）。

经验分享 & 注意事项

  • Cloudflare 不仅能搭建节点，还可用于搭建博客、图床、转发服务器等服务。
  • 建议使用 TLS 节点，启用加密传输，保障数据安全。
  • 如果客户端支持 分片（Fragment）功能，建议开启，可有效防止域名被阻断。
  • Cloudflare Pages 部署时，主 JS 文件必须命名为 _worker.js，如为 _worker (2).js 将无法识别。
  • 若首次部署时未绑定自定义域名，需要开启代理才能更新节点配置。
  • Pages 和 Workers 的节点生效时间有延迟，不是即时更新，建议绑定自定义域名后使用。
  • Cloudflare 访问自身托管内容时速度较慢，可结合「按量付费」机场使用，提升体验。

总结

通过 Cloudflare + VLESS 或 Trojan，可免费搭建不限流量、跨平台使用的节点服务，极大缓解网络流量焦虑，适合新手学习和使用。再配合域名、UUID 和客户端配置，短时间内即可快速上手。

如果你已经厌倦了寻找不稳定的免费节点，不妨试试自己搭建一个，安全、稳定、永久免费。
